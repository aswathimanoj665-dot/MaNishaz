# ✈️ Yahoo Airlines Ticket Reservation System

A simple airline ticket reservation webpage created using HTML and CSS. This project is designed to provide a basic user interface where passengers can enter their personal details and flight information before booking a ticket.

The main goal of this project is to practice front-end web development concepts such as HTML forms, CSS styling, and webpage design.

---

## 📖 Project Overview

The Yahoo Airlines Ticket Reservation System is a static web application developed as a mini project for learning purposes.

The application provides a booking form where users can enter customer information and travel details. Although it does not perform actual ticket booking, it demonstrates how an airline reservation interface can be designed using only HTML and CSS.

---

## 🎯 Objective

The objective of this project is to:

- Learn HTML form creation.
- Design a clean and user-friendly interface using CSS.
- Understand webpage structure and layout.

- Create a simple airline reservation webpage.
